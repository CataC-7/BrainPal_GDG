# **App Name**: RoutineRecall

## Core Features:

- Information Creation: Allow users input and save their either a 1) Daily workflow or 2) Emergency Intervention
- Routine Access: Enable users to easily access their saved daily workflows and emergency interventions.
- Homepage Display: Display a welcome text and two sections below, one that allow you to create (and has two parts: daily workflow and emergency intervention), and another that allows you to access (and has two parts: daily workflow and emergency intervention).

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to promote focus and clarity.
- Background color: Light, desaturated blue (#E3F2FD) for a clean, uncluttered feel.
- Accent color: Soft green (#A5D6A7) for positive reinforcement and to highlight important actions.
- Font: 'PT Sans', a humanist sans-serif, will be used for both headers and body, as it provides a balance of modernity and readability suitable for quick information access.
- Simple, intuitive icons to represent different types of routines or categories.
- Clean and organized layout with clear visual hierarchy for easy navigation.
- Subtle animations on routine access and updates to provide a smooth user experience.
